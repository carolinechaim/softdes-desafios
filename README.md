# softdes-desafios

## Descrição 
Esse código tem como principal objetivo ajudar novos alunos de programação a implementar os seus novos conhecimentos na resolução de desafios da linguaem Python. Esses desafios são criados pelos professores e reproduzidos no site para cada aluno ter acesso de forma individual. Ao conseguir resolver o desafio com sucesso, o aluno ganha uma pontuacao e pode seguir para o proximo disponivel
